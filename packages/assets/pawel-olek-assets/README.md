# @avatune/pawel-olek-assets

[![npm version](https://img.shields.io/npm/v/@avatune/pawel-olek-assets)](https://www.npmjs.com/package/@avatune/pawel-olek-assets)
[![npm bundle size](https://img.shields.io/npm/unpacked-size/@avatune/pawel-olek-assets)](https://www.npmjs.com/package/@avatune/pawel-olek-assets)

Pawel Olek style SVG assets for avatar generation.

## Description

This package provides SVG assets in pawel olek style for creating customizable avatars. Assets include various options for hair, eyes, eyebrows, mouth, nose, ears, head shape, and body/clothing.

## Installation

```bash
npm install @avatune/pawel-olek-assets
```

## Usage

### SVG Paths

```typescript
import { hair, eyes, mouth } from '@avatune/pawel-olek-assets';
```

### React Components

```typescript
import { HairShort, EyesBoring, MouthSmile } from '@avatune/pawel-olek-assets/react';
```

### Svelte Components

```typescript
import { HairShort, EyesBoring, MouthSmile } from '@avatune/pawel-olek-assets/svelte';
```

### Vue Components

```typescript
import { HairShort, EyesBoring, MouthSmile } from '@avatune/pawel-olek-assets/vue';
```
