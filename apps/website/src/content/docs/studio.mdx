Avatune Studio is a visual theme builder that lets you create custom avatar themes without writing code. Upload your SVG assets, position them visually, and export a ready-to-use theme package for React, Vue, Svelte, or Vanilla JS.

![Avatune Studio Preview](/studio-preview.gif)

## What You Can Do

- **Upload Custom Assets** - Import your own SVG designs for heads, eyes, mouths, hair, and accessories
- **Visual Positioning** - Drag and adjust elements to create perfect avatar compositions
- **Layer Management** - Control z-index and stacking order for complex designs
- **Multi-Framework Export** - Generate themes that work across all Avatune renderers
- **Live Preview** - See your avatar combinations in real-time as you build

## How It Works

1. **Upload Head Asset** - Start with your base head design as the foundation
2. **Add Categories** - Upload assets for eyes, mouths, hair, accessories, and more
3. **Preview & Adjust** - Fine-tune positioning, layering, and sizing for each element
4. **Export Theme** - Download your complete theme package ready for integration
