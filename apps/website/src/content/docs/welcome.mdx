# Welcome

Avatune helps you build beautiful, production-ready avatars with a single, cohesive toolkit. This documentation walks you through everything from the philosophy behind Avatune to implementation details across frameworks, themes, and predictive tools.

## Getting Around

- **Why Avatune** — Understand the core value prop and platform capabilities.
- **Framework Guides** — Learn how to install and use Avatune in React, Vue, Svelte, or Vanilla JS projects.
- **Themes** — Browse prebuilt themes like Yanliu, Miniavs, Flat, Micah, and Kyute, or learn how to customize your own.
- **Predictors** — Integrate AI-assisted generators such as hair color, hair length, and skin tone predictors.
- **Apps** — Explore ready-made app examples to see Avatune in action.

## Next Steps

Pick the guide that matches your current project, or jump straight into the packages directory to dive deep into a specific module. If you spot something that can be improved, contributions are welcome!
